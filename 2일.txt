HTML? 내용
<태그 속성="값">내용</태그>

CSS? 디자인
선택자 {속성: 값;}

선택자? CSS의 속성과 값을 HTML 요소에 지정하기 위해 사용한다.

선택자 종류
1. 요소(태그, 타입)
2. 아이디
3. 클래스

선택자 우선순위
아이디 > 클래스 > 요소

아이디와 클래스 이름 정의
1. 첫글자는 영문자, _, -
2. 두 번째 문자부터 영문자, _, -, 숫자
3. 영문의 대소문자는 구별한다.
4. 관례상 의미있는 내용으로 정의한다.

아이디와 클래스 차이점
1. 클래스는 한 페이지 내에서 중복 사용할 수 있으나 아이디는 같은 이름을 중복 사용할 수 없다.
2. HTML 클래스 속성 값은 하나 이상 지정할 수 으나 아이디는 지정할 수 없다.

CSS의 속성: 값;
color: red;
background-color: red;
font-weight: bold;
font-style: italic;
text-decoration: underline;
text-decoration: line-through;
font-size: 24px;
text-indent: 40px;
margin-left: 40px;

블록레벨요소: 
h1~h6, p, hr, blockquote, 
ul, ol, li,
dl, dt, dd

인라인요소:
b, strong, i, em, small, mark, sub, sup, del, ins, br, q
